# About Our Software
A Flutter App that allows users to discover, track, complete tasks about their hobbies, and share with the community. Users are able to create posts with photos, log their hobby level and XP, and browse activity from other users.

# How to Run Dev and Test Environment

```
flutter run
```

## Dependencies
- Flutter SDK: ^3.9.2
- Dart SDK: ^3.9.2
- firebase_core: ^4.5.0
- firebase_auth: ^6.2.0
- cloud_firestore: ^6.1.3
- image_picker: ^1.0.7

### Downloading Dependencies
1. Install Flutter from https://docs.flutter.dev/get-started/install — follow the guide for your OS (macOS, Windows, or Linux)
2. Install an IDE — [VS Code](https://code.visualstudio.com/) (recommended) or [Android Studio](https://developer.android.com/studio)
3. Install the Flutter and Dart extensions in VS Code (both free)
4. Clone this repository and open it in your IDE
5. Run the following command to install all packages:
```
flutter pub get
```
6. Set up Firebase for your platform by following the [FlutterFire setup guide](https://firebase.google.com/docs/flutter/setup)
